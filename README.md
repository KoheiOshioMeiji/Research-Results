**🍝 最高にうまいナポリタン([引用](https://mesisoku.blog.jp/archives/1203351.html))**

【材料（1人分）】

・茹でて一晩寝かせた太麺（2.2mm）…200g

・ケチャップ…お玉8～9分目（約70～80g）

・サラダ油…大さじ1

・ソーセージ…お好みの量（2～3本目安）

・マッシュルーム…お好みの量（スライス）

・タマネギ…1/4～1/2個（串切り）

・ピーマン…1/4個（5mm幅）

・※お好みで：粉チーズ、パセリ、ニンニク、マスタード、バター

【作り方】

① フライパンにサラダ油を熱し、ソーセージとマッシュルームを炒める
　→ 断面に軽く焼き目がつくまでしっかり加熱（中火で1分半～2分）

② タマネギを加え、しんなりするまで炒める（中火で1～1.5分）
　※この時点ではピーマンを入れない（香りと色を飛ばさないため）

③ ケチャップ（70〜80g）を投入し、中火で3分半～5分ほど炒める
　→ 以下の変化が現れたら“まとまった”状態
　✔ 色：鮮やかな赤 → レンガ色に変化
　✔ 水分：飛んで粘度が増し、ヘラでまとめて裏返せる
　✔ 音：じゅわじゅわ → パチパチ音に変化
　✔ 香り：酸味のある匂い → 甘く香ばしい匂いに変化
　※焦げないよう混ぜ続ける

④ ケチャップがまとまったらピーマンを加えて、30秒ほど軽く炒める
　→ 色と香りを生かすため加熱しすぎない

⑤ 火を弱めて、バターや好みのオイルを小さじ1ほど加える
　→ ソースを伸ばしてなじませ、そこへ寝かせた麺を投入

⑥ 強火に戻して麺を炒め、ソースをしっかり絡める（約1分半）
　→ ソースの濃い赤が麺と混ざり、食欲をそそるオレンジ色に変化！

⑦ 全体が熱々でソースが均一に絡んだら皿に盛る
　→ 粉チーズやパセリで仕上げれば完成！

【アレンジポイント】
・ニンニクを入れるなら → タマネギ投入の直前にみじん切りで
・マスタードを入れるなら → 麺投入後に小さじ1ほど加えると香り立ち◎

🍽️ ポイント総まとめ

麺は一晩寝かせた太麺でモチモチ感UP

ケチャップは徹底的に炒めて濃縮＆香ばしさUP

ピーマンは最後に入れて香りと色を生かす

「色・音・香り・粘度・返しやすさ」でケチャップ炒めの完成度を見極める
